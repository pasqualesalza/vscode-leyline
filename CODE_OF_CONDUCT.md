# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Instances of unacceptable behavior may be reported via
[GitHub Security Advisories](https://github.com/pasqualesalza/vscode-leyline/security/advisories)
or by contacting the maintainer through their
[GitHub profile](https://github.com/pasqualesalza).

Reports will be reviewed promptly and kept confidential.
